To program Arduino (note, this requires a bit of work, contact juliansl@samfundet.no if necessary)
01. Short the two plugs that you have to short.
02. Use Flip to program the arduino with the: Arduino-usbserial-uno.hex.
03. Plug out the arduino.
04. Plug in the arduino.
05. Configure the Arduino IDE to use a baudrate of 9600.
06. Edit the Arduino code.
07. Upload the Arduino code.
08. Short the two plugs that you have to short.
09. Use Flip to program the arduino with the: Arduino-keyboard-0.3.hex.
10. Plug out the arduino.
11. Plug in the arduino.
12. If it doesn't work goto 01 and curse Julian.


To program Autohotkey
1. Edit the pling.ahk file.
2. Compile the pling.ahk.
3. Add the pling.exe to a task that runs on startup/login in task scheduler.

To use the Arduino
1. Plug in the Arduino.
2. Add the pling.exe to a task that runs on startup/login in task scheduler.
